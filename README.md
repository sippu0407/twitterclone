# Twitter Clone Frontend

Twitter clone built using React, Redux.

Demo - [https://twitter-musk.vercel.app/#/](https://twitter-musk.vercel.app/#/)
## Features

1. Signup/Login
2. Add Tweet
3. Follow people
4. Like, Retweet, comment
5. View profile
6. Search users in explore section
7. Add bookmarks
8. Dark mode
